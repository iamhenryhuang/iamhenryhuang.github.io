3. Smart Trash Classification System: A Deep Learning Approach

Description：

Developing an image classification model using TensorFlow and Keras, leveraging transfer learning with the pre-trained InceptionV3 model to accurately classify waste.

Technical Details

  • Framework: TensorFlow and Keras
  • Model: InceptionV3 (pre-trained, fine-tuned last 15 layers)
  • Data Processing:
    • Dataset split into training and validation sets
    • Data augmentation: Shear, Zoom, Horizontal Flip
    • Resized input images to 128x128 pixels
  • Model Architecture:
    • Pre-trained InceptionV3 as feature extractor
    • Global Average Pooling layer
    • Fully connected Dense layer with 8 output units (softmax activation)
  • Optimization:
    • Loss Function: Categorical Crossentropy
    • Optimizer: RMSprop with a learning rate of 1e-5
  • Visualized predictions on test images
  • Performance Tracking: Integrated with TensorBoard
